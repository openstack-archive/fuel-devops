fuel-devops
===========

Django based application provides a sublayer between application and target
environment(all of supported by libvirt currently).

Uses PostgreSQL by default to store it's data.

Depencies
---------
 * postgresql
 * python-psycopg2
 * python-ipaddr
 * python-libvirt
 * python-virtualenv
 * python-ipaddr
 * python-paramiko
 * python-django (>= 1.4)
